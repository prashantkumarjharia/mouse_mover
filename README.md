# Mouse Movement Utility
Python based utility that moves cursor 10 pixels left and right with a 5 seconds delay until `Ctrl+C` is pressed.

## Requirements
- Python 3.6+
- pyautogui library

## Installation
Install the required dependencies:
```bash
pip install -r requirements.txt
```

## Usage
Run the Script using below command:
```bash
python main.py
```
> [!IMPORTANT]
> To stop the script, press `Ctrl+C` in the terminal.
